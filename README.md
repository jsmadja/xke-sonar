xke-sonar
=========

Prepare your stuff
------------------

Download sonar 3.2 at http://dist.sonar.codehaus.org/sonar-3.2.zip

Clone the exercices 
`git clone `

Unpack Sonar 3.2 
`unzip sonar-3.2.zip`

Start Sonar 
`sonar-3.2/bin/macosx-universal-64/sonar.sh start`

Download whole universe with Maven 
`./sonar-all.sh`

Now, you're ready to listen to me for a while :)
