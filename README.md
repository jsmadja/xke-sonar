Prepare your stuff
------------------

Download sonar 3.2
  http://dist.sonar.codehaus.org/sonar-3.2.zip

Clone the exercices
  git clone

Unzip sonar 3.2
  unzip sonar-3.2.zip

Start Sonar
  sonar-3.2/bin/macosx-universal-64/sonar.sh start

Download whole universe with Maven
